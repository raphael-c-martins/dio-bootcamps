# 🚀 Central de Projetos - DIO.me Bootcamps

Bem-vindo ao repositório centralizado para armazenamento e organização de todos os projetos práticos e desafios desenvolvidos durante os bootcamps da **DIO.me**.

Este repositório atua como um grande portfólio de estudos, onde cada pasta principal representa uma trilha ou bootcamp específico promovido em parceria com grandes empresas de tecnologia e do mercado.

## 📂 Diretórios de Bootcamps

Os projetos estão divididos por pastas. Abaixo, a lista dos bootcamps atualmente disponíveis no repositório:

- 🏦 **[Bootcamp Bradesco - GenAI, Dados e Cyber](./bootcamp-bradesco-genai-dados-cyber/)**
- 🍺 **[Bootcamp Heineken - IA e Dados](./bootcamp-heineken-ai-data/)**
- 🛡️ **[Bootcamp Santander - Cibersegurança](./bootcamp-santander_2-cybersecurity-challenges/)**

> 💡 **Dica:** Acesse a pasta de cada bootcamp para ler o seu respectivo `README.md` e entender os detalhes dos desafios, arquiteturas e tecnologias utilizadas em cada projeto.

## 🛠️ Sobre a Estrutura

Esta estrutura foi pensada para ser **simples e versátil**. Sempre que um novo bootcamp for iniciado ou um novo desafio for concluído, uma nova pasta será criada (ou atualizada) na raiz deste repositório, garantindo que todo o aprendizado fique consolidado em um único lugar.
